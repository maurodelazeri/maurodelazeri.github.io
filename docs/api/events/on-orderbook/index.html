OnOrderbook

#OnOrderbook

The Orderbook event is generated when a change occurs in any of the 40 levels of the orderbook. It is full-duplex bidirectional streams and processed by OnTrade function.

Parameters:

There is no required parameters but you can filter the stream specifing a structure OrderbookRequest
